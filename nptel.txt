Unit 1: Introduction to OOP  [Lectures 1–5]
  Topics: procedural vs OOP paradigm, OOP features (modularity, reusability,
  encapsulation, inheritance, abstraction, polymorphism), objects and classes,
  class as blueprint, state and behavior of objects, historical overview of OOP,
  programming paradigms comparison, benefits of OOP, real-world analogies
  (animals, pen, bank account).

Unit 2: Classes and Objects in C++ and Java  [Lectures 2–7]
  Topics: class definition syntax in C++ and Java, member data vs member
  functions (methods), access specifiers (public, private, protected), object
  instantiation, dot operator, static vs dynamic object creation (new keyword),
  class declaration vs definition, scope resolution operator (::), 'this' keyword
  scope, in-class and out-of-class method definitions.

Unit 3: Constructors and Access Specifiers  [Lectures 8–11]
  Topics: default constructor, parameterized constructor, copy constructor,
  constructor overloading, destructor (~), deep copy vs shallow copy, constructor
  chaining, access specifiers in C++ and Java (public/private/protected/default),
  constructor in Java vs C++, 'this' keyword usage in constructors,
  dynamic object creation with new/delete.

Unit 4: Inheritance  [Lectures 12–15]
  Topics: single inheritance, multilevel inheritance, multiple inheritance,
  hierarchical inheritance, hybrid inheritance, base class and derived class,
  'extends' keyword in Java, colon syntax in C++, access control in inheritance,
  method overriding basics, super keyword in Java, constructor calling in
  inheritance chain, diamond problem, virtual base class to solve diamond problem,
  friend function and friend class concept.

Unit 5: Polymorphism — Overloading and Overriding  [Lectures 16–20]
  Topics: compile-time vs runtime polymorphism, function overloading,
  operator overloading in C++, constructor overloading, 'this' keyword in C++
  (returning *this), method overloading in Java, method overriding in Java and C++,
  @Override annotation, dynamic dispatch, upcasting and downcasting,
  static binding vs dynamic binding, covariant return types.

Unit 6: Encapsulation and Data Abstraction  [Lectures 21–25]
  Topics: encapsulation definition and implementation, getters and setters,
  data hiding, tight coupling vs loose coupling, data abstraction vs encapsulation,
  abstract class in C++ (pure virtual functions), abstract class in Java
  (abstract keyword), interface in Java (100% abstraction), virtual functions,
  vtable and vptr, pure virtual function (= 0 syntax), abstract vs concrete class,
  difference between abstract class and interface.

Unit 7: Exception Handling  [Lectures 26–30]
  Topics: types of exceptions (compile-time vs runtime), try-catch-throw in C++,
  try-catch-finally in Java, multiple catch blocks, catch block ordering
  (specific to general), nested try blocks, custom/user-defined exceptions,
  std::exception class hierarchy, checked vs unchecked exceptions in Java,
  'throws' keyword in Java, 'throw' vs 'throws', finally block guarantee,
  rethrowing exceptions, stack unwinding.

Unit 8: File Handling and Serialization  [Lectures 31–35]
  Topics: file I/O in C++ (ifstream, ofstream, fstream), file I/O in Java
  (FileReader, FileWriter, BufferedReader, Scanner), file modes (read/write/
  append/binary), character reading, line reading, CSV file handling, binary
  files, serialization and deserialization in Java (Serializable interface,
  ObjectOutputStream, ObjectInputStream), transient keyword, file pointer
  operations (seekg, seekp, tellg, tellp), error handling in file operations.

Unit 9: Templates and Generics  [Lectures 36–40]
  Topics: function templates in C++, class templates in C++, template
  specialization, typename vs class keyword in templates, type safety,
  code reusability with templates, generics in Java (type parameters T/E/K/V),
  bounded type parameters (extends), wildcard (?), generic methods, raw types,
  type erasure in Java, generics in Python (TypeVar, Generic class),
  difference between C++ templates and Java generics.

Unit 10: Standard Template Library (STL)  [Lectures 41–45]
  Topics: STL components (containers, iterators, algorithms), sequence containers
  (vector, list, deque, array), associative containers (map, set, multimap,
  multiset), unordered containers (unordered_map, unordered_set), container
  adapters (stack, queue, priority_queue), iterators (begin/end, forward/
  backward/random access), STL algorithms (sort, find, count, transform),
  auto keyword with iterators, pair and make_pair, emplace_back vs push_back.

Unit 11: Design Patterns  [Lectures 46–50]
  Topics: creational patterns (Singleton, Factory Method, Abstract Factory),
  structural patterns (Adapter, Decorator, Facade), behavioral patterns
  (Observer, Strategy, Iterator), Singleton pattern (private constructor,
  static instance, thread safety), Factory pattern (object creation without
  specifying class), Observer pattern (subject-observer relationship, event
  notification), design pattern categories, SOLID principles context,
  UML representation of patterns, real-world use cases.

Unit 12: Advanced Topics — Multithreading and Case Studies  [Lectures 51–60]
  Topics: multithreading concepts (thread, process, concurrency), thread
  creation in Java (Thread class, Runnable interface), thread lifecycle
  (new/runnable/running/blocked/terminated), synchronization (synchronized
  keyword, locks, semaphores, monitors), deadlock (causes: mutual exclusion,
  hold-and-wait, no-preemption, circular wait), deadlock prevention strategies,
  race condition, network programming basics (socket, ServerSocket in Java),
  HTTP communication, case studies (Library Management, Hotel Reservation,
  Shopping Cart, Employee Payroll — applying OOP concepts end-to-end).
